== Description ==

The completely redeveloped Social Sharing Buttons! Now with FontAwesome for even more customization and flexibility! Add social sharing to all your WordPress postings. Select from a list of popular social media icons including FaceBook, Pinterest, Tumblr and more! Complete control over icon colours and backgrounds. Increase the size of the icons and make them sticky or embedded.

**Features**
 * FontAwesome integration for clearest image
 * Icons can be any colour
 * Resize icons to any size
 * Backgrounds can be customized to any colour
 * Circle or Square backgrounds
 * Add it as a floating bar
 * Place it above or below you content

**Social Networks**
 * Facebook
 * Twitter
 * LinkedIn
 * StumbleUpOn
 * Google Plus
 * Tumblr
 * Pinterest
 * Reddit
 * Digg

 * Xing
 * Email
 * Print

Let us know!


== Installation ==

This section describes how to install the plugin and get it working.

1. Extract the zip file and drop the contents into the wp-content/plugins/ directory of your WordPress installation.
2. Activate the plugin from the plugins page.
